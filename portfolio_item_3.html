Marvel Comics Landing Page

Marvel Comics is the brand name and primary imprint offering access to over 25,000 digital comics including fan favorite characters like Spiderman, Captain America, Ironman, Thor, and more.


OVERVIEW

The goal for this mock project was to create a simple and straightforward digital comic book service site for users to navigate. Aside from design, responsiveness and utilizing javascript were top priority in this project.

CHALLENGES

The only challenge that I had with this project aside from getting the javascript working was the tab switch reveal. I wanted to make sure site visitors knew exactly which tab they were under so I incorporated a little bit of javascript to make a red bar underneath the selected icon.


THE SOLUTION

I wanted to draw users into the site as soon as they landed on the page so I opted for using a hero image to draw in viewers. I researched Netflix's landing page asthetic and incorporated it into this project to give it a modern feel with comic book characters as the main focus. The solution I came up with to display the pricing data was to use a table and a simple tab switch to show the offers.
